    Comprehensive Review: Tutorials and Practice Tasks

    Date: October 2025 Reviewer: Claude Code Files Reviewed: 11 tutorial guides, 15 practice tasks

    Executive Summary

    The tutorials and practice tasks are high quality with clear pedagogical goals, excellent use of technology (QR codes, interactive elements), and strong connections to student learning and assessment.

    Critical issues fixed: - ✅ Broken link in stage4_wk4.qmd email template - ✅ Broken link in speaking-feedback-demo.qmd - ✅ Author name typo in writing-concisely-abstracts.qmd - ✅ Incomplete subtitle in studying-using-psychology-to-be-better-psychologist.qmd - ✅ Missing “Tutor tasks” section in stage_2_wk3.qmd

    Overall assessment: With the above fixes, materials are in excellent shape and ready for use.

    Key Findings

    Strengths

    1. Excellent pedagogical design
      • Clear learning objectives stated upfront
      • Tasks connect explicitly to assessment (especially stage1_5.qmd)
      • Good scaffolding from simple to complex tasks
      • Mix of individual, paired, and group activities
    2. Technology integration
      • QR codes for phone-friendly access to materials
      • Interactive JavaScript elements (e.g., values checklist)
      • Touch-friendly checkboxes for mobile use
      • Consistent Psybot integration for feedback
    3. Quality of materials
      • Real-world examples and scenarios
      • Research-based approaches (e.g., FIT imagery exercise)
      • Clear, undergraduate-level language
      • Good balance of guidance without being prescriptive

    Areas for Improvement

    1. Consistency in formatting
      • Some tutorials lack time estimates (now noted below)
      • Instructional language mostly consistent (“Ask students…”) with minor variations
      • Overview sections present in most but not all tasks
    2. Clarification needed
      • stage1_5_extension_draft.qmd – unclear if this is supplementary material or outdated
      • A few HTML comments (TODOs) suggest incomplete planning

    Suggestions for New Content

    New Stage 1 Critical Thinking Tutorials

    1. “Evaluating Evidence Quality” - Target: Stage 1, Week 7-8 - Focus: Hierarchy of evidence (systematic reviews > RCTs > correlational > case studies) - Activities: - Rank different types of evidence for a given claim - Practice identifying study designs from abstracts - Discuss when different designs are appropriate - Links to: PSYC421 research methods, report writing - Materials needed: - Set of 6-8 abstracts with different designs - Decision tree for choosing research designs - Student handout with ranking exercise

    2. “Understanding Effect Sizes and Practical Significance” - Target: Stage 1, Semester 2 (after hypothesis testing) - Focus: Difference between statistical and practical significance - Activities: - Interpret effect sizes from published studies - Decide which findings are “meaningful” vs “trivial” - Practice reporting results with context - Links to: PSYC421 results sections, PSYC424 statistics - Materials needed: - Real published results with varying effect sizes - Scenarios where students judge practical importance - Guide to interpreting Cohen’s d, r, etc.

    3. “Pre-registration and Research Transparency” - Target: Stage 1, late semester 2 - Focus: Why researchers pre-register studies, p-hacking, transparency - Activities: - Interactive demonstration of p-hacking - Discussion of researcher degrees of freedom - Review a simple pre-registration template - Links to: Open science, research integrity, dissertation planning - Materials needed: - Dataset that can be analysed multiple ways - Examples of pre-registered vs non-pre-registered studies - Simple pre-registration template

    New Writing Tutorials

    4. “Hedging Your Claims: Academic Tone and Certainty” - Target: Stage 1 or 2 - Focus: Appropriate use of tentative language in psychology - Activities: - Identify over-confident vs appropriately hedged claims - Practice re-writing claims with appropriate certainty - Understand when to use “suggests”, “indicates”, “demonstrates” - Links to: All essay writing, discussion sections - Practice task: Edit paragraph to improve hedging - Materials needed: - Examples of poorly hedged claims - Guidelines for choosing appropriate language - Practice paragraphs to edit

    5. “Structuring Arguments: From Claim to Evidence” - Target: Stage 2 - Focus: Building persuasive academic arguments - Activities: - Identify claims vs evidence in sample paragraphs - Practice the PEEL structure (Point, Evidence, Explanation, Link) - Reverse-engineer strong paragraphs from model essays - Links to: Essay writing across all modules - Practice task: Write structured paragraphs on given topics - Materials needed: - Model paragraphs with clear vs unclear structure - PEEL writing framework - Topics for practice writing

    New Stage 4 Career/Workplace Tutorials

    6. “Translating Your Psychology Skills for Employers” - Target: Stage 4, early semester - Focus: How to articulate psychology skills in non-academic contexts - Activities: - Map dissertation skills to job requirements - Practice describing research in plain language - Create “skills translation” for CVs and applications - Outcome: Students can explain transferable skills - Materials needed: - Sample job descriptions (varied sectors) - Skills translation template - Examples of good/poor skill descriptions

    7. “Building Your Professional Network” - Target: Stage 4, mid-semester - Focus: Networking for introverts, LinkedIn, informational interviews - Activities: - LinkedIn profile review (paired activity) - Practice “elevator pitch” for their research - Plan informational interview with alumnus - Outcome: Students have actionable networking plan - Materials needed: - LinkedIn profile checklist - Elevator pitch framework - Alumni contact list with psychology careers - Email template for informational interviews

    8. “Psychology Outside Academia: Alternative Career Paths” - Target: Stage 4, late semester - Focus: Diverse careers using psychology (UX, HR, consulting, etc.) - Activities: - Guest speaker or video from psychology graduate in non-clinical role - Small group research on specific career path - Create career pathway map showing steps from graduation to role - Outcome: Students see breadth of career options - Materials needed: - Guest speaker or recorded interviews - Career pathway templates - Resources: Prospects, BPS careers pages, LinkedIn alumni

    9. “Portfolio Preparation: Showcasing Your Work” - Target: Stage 4, semester 2 - Focus: Creating professional portfolio of psychology work - Activities: - Identify strongest work samples (reports, presentations, data visualisations) - Practice writing project descriptions for non-academic audiences - Create simple portfolio website or PDF - Outcome: Students have shareable portfolio - Materials needed: - Portfolio examples from previous students - Templates for describing projects - Technical guidance (simple website builders, PDF creation)

    10. “Understanding Postgraduate Options: MSc, PhD, DClinPsy” - Target: Stage 4, early semester - Focus: Different postgraduate routes, application timelines, funding - Activities: - Compare different programmes (clinical, research, educational, forensic) - Timeline for applications (with key dates) - Funding sources and application strategies - Outcome: Students understand pathways and requirements - Materials needed: - Programme comparison chart - Application timeline template - Funding resources list - Tips from current postgrad students

    New Study Skills Tutorials

    11. “Managing Cognitive Load: Study Strategies for Complex Material” - Target: Stage 2 - Focus: Working memory limitations, chunking, interleaving - Activities: - Demonstration of working memory limits - Practice chunking complex information - Create interleaved study schedule - Links to: Cognitive psychology content, exam preparation - Practice task: Redesign study notes using chunking principles - Materials needed: - Working memory demonstration exercises - Examples of chunked vs unchunked information - Study schedule template

    12. “Retrieval Practice: Testing Yourself Effectively” - Target: Stage 1, late semester 1 - Focus: Testing effect, creating effective practice questions - Activities: - Try different types of retrieval practice - Create practice questions for recent lectures - Compare recognition vs recall tasks - Links to: All exam preparation, PSYC424 memory content - Practice task: Create question set for upcoming exam - Materials needed: - Examples of good/poor practice questions - Different question formats (MCQ, short answer, essay prompts) - Guidance on spacing retrieval practice

    Instructional Language Standards

    Use consistently: - “Ask students…” (for questions to pose) - “In pairs…” / “In small groups…” / “As a whole group…” (for activity groupings) - “Whole group discussion…” (for plenary sharing) - “Discuss…” / “Share…” / “Explain…” (for student actions)

    Time estimates: - Always include approximate times: “(5 minutes)”, “(10-15 minutes)” - Include time checks for longer tutorials: “Timecheck: Aim to finish this task halfway through”

    Student-facing instructions: - Use bold for questions: “What factors might influence memory?” - Use blockquotes for text students should read: > All other things being equal...

    Materials sections: - Always include both link and QR code - Format: [Descriptive text](filename.html) then QR code image - QR code width: 50% for supplementary materials, 80% for primary materials
